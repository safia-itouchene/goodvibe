## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- A modern web browser

### ⬇️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/safia-itouchene/goodvibe.git
   cd goodvibe
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```

4. **Open in your browser:**
   Navigate to `http://localhost:3000`

---

## 💻 Usage

Simply open the application and start spreading good vibes! The interface is self-explanatory and intuitive.

```bash
# Development
npm start

# Build for production
npm run build

# Run tests
npm test
```
